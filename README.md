# 🛡️ Smart ToS Decoder

An AI-powered web application that simplifies complex Terms & Conditions into easy-to-understand summaries.

## 🚀 Features

- 🤖 AI-powered Terms & Conditions analysis (Google Gemini)
- 📊 Dynamic Risk Score (0–100)
- 📈 Interactive Risk Chart
- 🔍 Risk Signal Detection
- 🔐 Permission Detection
- 💡 AI Recommendations
- 📜 Analysis History
- 🌙 Dark Mode UI
- 📱 Fully Responsive Design

## 🛠️ Tech Stack

### Frontend
- React
- Axios
- CSS3

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Google Gemini API

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-ToS-Decoder.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
GEMINI_API_KEY=your_gemini_api_key
```
